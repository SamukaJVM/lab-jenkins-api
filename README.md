# CI/CD (Integração Contínua e Entrega Contínua)

Um projeto laboratório CI/CD com Jenkins desenvolvido com Spring Boot 4.0.3, Java OpenJDK 17 e PostgreSQL.

---

## 🎯 Sobre o Projeto

Sistema backend para controle financeiro com:
- ✅ API REST com Spring Boot
- ✅ Banco de dados PostgreSQL
- ✅ Pipeline CI/CD automatizada
- ✅ Análise de código com SonarQube
- ✅ Observabilidade (Prometheus + Grafana)

## 📦 Pré-requisitos

- Sistema Operacional Linux Ubuntu 24.04.4 LTS
- Instalar Openjdk 17.0.18
- Instalar Apache Maven 3.8.7
- Instalar Git version 2.43.0
- Instalar IntelliJ IDEA
- Instalar Docker Engine 29.3.0
- Instalar Jenkins 2.541.2
- Instalar K3d v5.8.3

## 🚀 Tecnologias

**Backend:**
- Java OpenJDK 17.0.18
- Spring Boot 4.0.3
- PostgreSQL 17
- Apache Maven 3.11.0

**DevOps:**
- Docker engine 29.3.0
- Jenkins 2.541.2
- SonarQube 9.0-community
- Kubernetes K3d 5.8.3
- Prometheus e Grafana
